
### Simple example of UI Testing Framework using Java, Maven, TestNg, Selenium.

#### Clone project:
```
git clone https://github.com/vsshk/java-maven-testng-selenium-example-simple.git
cd java-maven-testng-selenium-example-simple
```
### How to run
```
$ mvn  test
```
